# Logo Files

Place the following logo images in this directory:

1. **bible-logo.png** - Bible/Hymn book icon (green with red cross)
   - Used in HomeScreen header
   - Recommended size: 60x60px

2. **music-ministry-logo.png** - Music Ministry logo (musical note design)
   - Used in HomeScreen header
   - Recommended size: 100x40px (or maintain aspect ratio)

These images will be displayed in the app header and home screen.
